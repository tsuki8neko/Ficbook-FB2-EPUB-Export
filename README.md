# Ficbook FB2 / EPUB Export

Скрипт позволяет скачивать тексты с Ficbook и экспортировать их в форматы **EPUB** и **FB2**.

Проект распространяется в виде userscript’а и предназначен для локального использования.

---

## Возможности

- экспорт текстов с Ficbook
- поддержка форматов **EPUB** и **FB2**
- сохранение структуры текста
- генерация оглавления с ссылками
- созранение информации из шапки произведения

---

## Установка (Userscript)

1. Установи менеджер userscript’ов:
   - **Tampermonkey** (Chrome, Edge)
   - **Violentmonkey** (Firefox)

2. Перейди в раздел **Releases**:
   https://github.com/tsuki8neko/Ficbook-FB2-EPUB-Export/releases

3. Скачай последний `.user.js` файл и установи его в менеджер скриптов.

---

## Использование

1. Открой страницу произведения на Ficbook с содержанием https://ficbook.net/readfic/*
2. После загрузки страницы в правом нижнем углу появятся кнопки экспорта
3. Выбери нужный формат:
   - EPUB
   - FB2
4. Файл будет сгенерирован и сохранён локально
